Dictionary Generator

Make customized wordlists for penetration testing practice
(brute force attack, dictionary attack, etc.). It is possible
to generate different wordlists using different methods, choose
your favourite one and enjoy your wordlist! All the information
provided on this repository are for educational purposes only.
The repository and author of the repository is no way responsible
for any misuse of the information.

Keys for data entry automation
	-0 - numbers from 0 to 9
	-a - small latin letters
	-A - big latin letters
	-! - symbols ! " # $ ...
