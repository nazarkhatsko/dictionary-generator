Dictionary Generator

Make customized wordlists for penetration testing practice (brute force attack, dictionary attack, etc.). It is possible to generate different wordlists using different methods, choose your favourite one and enjoy your wordlist! Keys for data entry automation

  • -0 - numbers from 0 to 9
  • -a - small latin letters
  • -A - big latin letters
  • -! - symbols ! " # $ ...
